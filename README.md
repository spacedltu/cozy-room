# Pinterest Shop System

## Kaip naudoti

1. Įkelk visus failus į GitHub repository:
   - index.html
   - style.css
   - script.js
   - data.json
   - minimal-cozy-room folder

2. Nuotraukos:
   - hero: minimal-cozy-room/hero.jpg
   - produktai: minimal-cozy-room/1.svg iki 6.svg

3. Kai turėsi tikras produktų nuotraukas:
   - įkelk jas į minimal-cozy-room folder
   - pavadink: 1.jpg, 2.jpg, 3.jpg, 4.jpg, 5.jpg, 6.jpg
   - data.json pakeisk image laukus į:
     minimal-cozy-room/1.jpg
     minimal-cozy-room/2.jpg
     ir t. t.

4. Produktų tekstus, kainas ir Amazon affiliate linkus keisk tik data.json faile.

5. GitHub Pages URL:
   https://spacedltu.github.io/cozy-room/
